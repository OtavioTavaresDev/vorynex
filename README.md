## 🛠️ Customization

- **Replace logo:** Update `assets/images/vorynex.png` with your own
- **Change language strings:** edit `data/translations.json`
- **Add/remove products:** edit `data/products.json`
- **Update contact numbers:** modify `WHATSAPP_NUMBER` in `assets/js/main.js`
- **Colors & fonts:** tweak CSS variables in `assets/css/style.css`

## 📦 Deploy to GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Select branch `main` and folder `/ (root)`.
4. Your site will be live at `https://<username>.github.io/vorynex/`.
